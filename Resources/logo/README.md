# 正式名称
Mac版のアプリの正式名称は「JW-CAD for Mac」です。「Jwcad for mac」や「Jw_cad for Mac」といった記載をしないでください。

また、本家（Windows版）は「Jw_cad」と記載するルールにしています。しかし「Jww」「Jw_win」など、名称が統一されていないため検索対象用に記載したい場合は、「JW_cad（Jww,Jw_win）」といった記載をしてください。
